# Regulartory Motif Finding 
```
usage: motif_finding.py [-h] [--input INPUT] [--k K] [--start START]
                        [--end END] [--optimization OPTIMIZATION]

Regulartory Motifs

optional arguments:
  -h, --help            show this help message and exit
  --input INPUT         file of sequences
  --k K                 k-mer length
  --start START         starting sequence
  --end END             ending sequence
  --optimization OPTIMIZATION
                        sets optimization level [0= bruteforce, 1=
                        branchandbound
```
